# Autonomous-Alcohol-using-mq2-555timer
Mini hardware project: Alcohol detection system using MQ2 gas sensor, 555 timer, BC547 and relay to trigger buzzer/LED alerts above a set threshold for vehicle and workplace safety.

Overview:It detects alcohol vapour and activates buzzer/LED using MQ2 + 555 timer + relay for safety applications (drunk‑driving prevention, workplace monitoring, etc.).​

Features:

Detects alcohol vapours using MQ2 sensor.​

Threshold‑based audio/visual alert with buzzer/LED and relay.​

Adjustable sensitivity via preset for different environments.​

Hardware Used:

MQ2 sensor, 555 timer IC, BC547 transistor, relay module, buzzer/LED, 7805 regulator, power supply.
